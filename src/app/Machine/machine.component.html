Neural Machine Translation

Developed a Sequence to Sequence Recurrent Neural Network using attention mechanism to translate the human readable dates (25th of July, 2018) to machine readable dates (2018-07-25). The input sequence is converted to one hot vector and then it is passed into the Bi-directional LSTM as we need to know the entire sentence to learn, then this output is exposed to the attention mechanism post which it is passed to another layer of uni-directional LSTM and then softmax function is applied to get the required output.

Trigger Word Detection

Developed a Recurrent Neural Network model which takes an audio clip containing the trigger word as an input and whenever it comes across the trigger word it triggers a custom event. The dataset was generated by inserting the trigger word randomly in any random audio clip.The inputs are passed through 1D convolution post which it is passed through GRU units. Batch Normalization and Dropout has been applied as Regularization methods to help improve the accuracy.

Conversational Chatbot

Designed and developed a conversational chatbot using the sequence to sequence model of a Recurrent Neural Network and applying the concepts of Deep NLP. This chatbot has been trained on the movie corpus dataset. In this model, the input data has been preprocessed by tokenization and converting the sentences into the embeddings vector which makes it easier for the neural network to train. The sequence to sequence model has been implemented using the bi-directional LSTM network and attention mechanism along with dropout regularization. Attention mechanism has also been applied during the decoding process of the model. Adam optimizer has been used for optimizing the loss function.

Uncertainty Quantifier

Designed and developed a machine learning model which takes the salmon tool output of the transcripts data as an input and converts the faulty transcripts in the data to non-faulty transcripts. Preprocessed the data and then analyzed the various features in the data so that only useful features can be included in the data to improve the accuracy and then generated a new dataset on which the SVM classification model was applied. Based on the output of the model, the feature values were adjusted to make the faulty transcripts, non-faulty and this process was continued until the faulty transcript count was reduced by 30%.
