VirtualCoin

Designed and developed a cryptocurrency similar to bitcoin following the bitcoin white paper and wrapped an api around it using nodejs and express framework along with end-to-end testing. It implements the three main features of a cryptocurrency which are a secure blockchain, wallet and mining. Implemented the proof of work algorithm as the consensus algorithm for the cryptocurrency. Implemented the peer-to-peer server for the other nodes to join the network and transfer cryptocurrency to each other and also the miners to join the network and mine the transactions and add them to the blockchain by solving the proof of work algorithm.

Voting

Designed and Developed an end-to-end decentralized application which uses the blockchain technology to register the voters and cast their vote. This application uses the modifiers to ensure that only the chairperson or the organizer of the voting process can register the candidates, to ensure that no one is able to register after the registration period ends, to ensure that nobody can vote until they are registered and no winner is declared until the voting process is done. Implemented the unit-tests to ensure the correct functionality of the application. Developed the front end in Angular and the backend using the Javascript and Smart Contracts.
